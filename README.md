# 小兔仙儿 基于uniapp与黑马程序员提供API的手机平台的电商项目
## 目前着重兼容的平台
- h5
- weixin

 *打包到别的平台有风险*
## 需要安装的平台依赖
- node
- npm
### 安装包
`
npm i
`
## 运行

```\n npm run dev:h5\n```
```\n npm run dev:mp-weixin\n```
