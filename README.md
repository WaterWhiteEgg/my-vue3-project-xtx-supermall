# 小兔仙儿 基于uniapp与黑马程序员提供API的手机平台的电商项目
## 目前着重兼容的平台
- h5
- weixin

 *打包到别的平台有风险*
## 需要安装的平台依赖
- node
- npm
- 微信平台 需要额外安装 `微信小程序工具` 解析
- 安卓平台 需要额外安装识别安卓的编译器解析以及打包 可选择 `HBuilderX`
- ios平台 需要额外安装识别ios的编译器解析以及打包 可选择 `HBuilderX`
- 其他小程序 安装对应的工具 *我都不知道有什么小程序和工具;-)*
### 安装包
`
npm i
`
## 运行

```npm run dev:h5```  

or  

```npm run dev:mp-weixin```
